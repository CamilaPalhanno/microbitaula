1. Introdução ao Micro:bit (10 minutos)

O Micro:bit é um pequeno computador programável desenvolvido pela BBC para incentivar crianças a aprenderem programação e eletrônica. Lançado em 2015, o dispositivo foi projetado para ser acessível e fácil de usar, tornando-o uma ferramenta ideal para a educação.

2. Programação em Blocos com MakeCode (20 minutos)

O MakeCode é uma plataforma de programação visual que permite criar programas para o Micro:bit usando blocos. É uma maneira intuitiva e acessível de introduzir conceitos de programação para alunos de todas as idades.

3. Atividade Prática Sugerida (25 minutos)

Vamos criar um "Medidor de Humor" usando o Micro:bit. Este projeto utiliza os botões e a matriz LED para permitir que os alunos expressem seu humor de forma interativa.

Passos para o "Medidor de Humor":

  1. Use o botão A para mostrar um rosto feliz na matriz LED.
  2. Use o botão B para mostrar um rosto triste.
  3. Ao pressionar ambos os botões, mostre um rosto neutro.

Tente implementar isso no simulador MakeCode acima!
